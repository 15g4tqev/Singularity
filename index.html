Как зарегистрироваться на хакатон?

Для регистрации на хакатон необходимо заполнить форму на сайте, указав свои контактные данные и информацию о команде.

Какие требования к участникам?

Участники должны иметь базовые знания в области блокчейн-технологий и программирования. Команды могут состоять из 2-5 человек.

Какое оборудование нужно для участия?

Для участия необходим компьютер с доступом в интернет и установленным необходимым программным обеспечением для разработки.

Правила соревнований

1. Общие положения

1.1. Настоящие правила регулируют порядок проведения хакатона Singularity v1.

1.2. Участники обязуются соблюдать все правила и требования организаторов.

2. Условия участия

2.1. К участию допускаются команды от 2 до 5 человек.

2.2. Каждый участник может быть членом только одной команды.

3. Требования к проектам

3.1. Проекты должны соответствовать выбранному направлению.

3.2. Все решения должны быть реализованы в рамках хакатона.

1. Сбор информации

1.1. Мы собираем только необходимую информацию для участия в хакатоне.

1.2. Все данные обрабатываются в соответствии с законодательством.

2. Использование данных

2.1. Ваши данные используются только для организации мероприятия.

2.2. Мы не передаем информацию третьим лицам.

3. Безопасность

3.1. Мы принимаем все необходимые меры для защиты ваших данных.

3.2. Доступ к информации имеют только уполномоченные лица.
